# typescript-express

## Getting started

### 1. Clone project

```bash
git clone https://github.com/bkkoli/typescript-express.git
```

### 2. Install dependencies

```bash
cd typescript-express && npm i
```


### 3. Required Service Preparation

```
npm run setting:start
```

### 4. Start server

```bash
npm run start
```
