# signage-server

## Getting started


### 1. Clone project
```bash
git clone https://github.com/bkkoli/typescript-express.git
```

### 2. Install dependencies
```bash
cd typescript-express && npm i
```

### 3. Modify your env file
- dev.env : developement
- prod.env : production
- test.env.js : for Jest

### 4. Start server
```bash
npm run start
```
